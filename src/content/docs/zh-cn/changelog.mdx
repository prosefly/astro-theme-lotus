## 未发布

- 将 theme config 中的 `navbar` 重命名为 `siteNav`，`sidebars` 重命名为
  `docsNav`。
- 添加自动读取 `theme.config.json`，并在 `/schema.json` 发布 JSON Schema；
  传给 `lotus({...})` 的选项会覆盖 JSON 中的值。
- 将语言选择器移动到站点品牌旁边，并从移动端菜单界面中移除。

## [0.5.0](https://github.com/prosefly/astro-theme-lotus/releases/tag/0.5.0)

<Badge variant="subtle" color="neutral">2026-07-21</Badge>

- 将 `@prosefly/astro-components` 更新到 `^0.4.0`，并将 Prosefly 组件
  token 和 class 前缀从 `pl` 迁移到 `pf`。
- 添加 `lotus-prose` 尺寸变体，并支持在正文容器上使用
  `text-sm`、`text-base` 和 `text-lg`。
- 导出 `DocsLayout`，作为带主内容区和右侧栏插槽的可复用文档外壳布局。
- 导出可复用的 `Sidebar` 主题组件，方便自定义页面和模板使用。
- 添加 `markdown.cjkFriendly`，配置中文、日文或韩文语言环境时会自动启用。
- 构建侧边栏导航时支持解析基于 Promise 的侧边栏项目组。
- 修复侧边栏滚动到顶部或底部后无法继续带动页面滚动的问题。
- 将 `@prosefly/astro-components` 更新到 `^0.4.1`，并将 `Badge` 圆角映射到
  当前 Lotus 圆角尺度。
- 收窄根包导出，只保留已文档化的集成和主题配置 API。

## [0.4.0](https://github.com/prosefly/astro-theme-lotus/releases/tag/0.4.0)

<Badge variant="subtle" color="neutral">2026-07-19</Badge>

- 启用来自 `@prosefly/astro-components` 的 Prosefly 提示块指令语法。
- 将 `@prosefly/astro-components` 更新到 `^0.3.0`。
- 将 Markdown 相关集成选项收拢到 `markdown` 下；移除顶层
  `expressiveCode`、`calloutDirectives` 和 `packageManagerTabs`。
- 添加 `themeModeControl`，无需覆盖 `ThemeSwitch` 即可选择内置主题模式控件。
- 修复独立 `SiteHeader` 在搜索和 assistant 触发器都关闭时，移动端菜单没有靠右的问题。

## [0.3.1](https://github.com/prosefly/astro-theme-lotus/releases/tag/0.3.1)

<Badge variant="subtle" color="neutral">2026-07-19</Badge>

- 添加 `llms` 配置，用于关闭 `/llms.txt` 或生成 `/llms-full.txt`。
- 只有启用 Pagefind 搜索提供器时，才输出 Pagefind 索引属性。

## [0.3.0](https://github.com/prosefly/astro-theme-lotus/releases/tag/0.3.0)

<Badge variant="subtle" color="neutral">2026-07-19</Badge>

- 为 `h2`、`h3` 和 `h4` 添加 SSR 标题锚点。
- 为自定义页面添加基于 Shiki 高亮的 `CodeBlock` 导出。
- 为自定义页面添加 `Prose` 导出。
- 为文档页面元数据添加 Markdown 替代链接。
- 添加自动生成的 `/llms.txt` 端点，用于供 AI 读取的文档发现。
- 为 `info`、`success`、`warning` 和 `danger` 添加语义化状态颜色 token。
- 将 `@prosefly/astro-components` 更新到 `0.2.1`。
- 将 `SiteHeader` 重构为独立站点页眉。
- 修复标题内正文链接的样式。
- 改进移动端标题锚点样式。
- 修复宽屏下文档内容宽度，避免页面操作与目录重叠。

## [0.2.0](https://github.com/prosefly/astro-theme-lotus/releases/tag/0.2.0)

<Badge variant="subtle" color="neutral">2026-07-18</Badge>

- 添加繁体中文、日文、韩文、法文、德文、西班牙文、巴西葡萄牙文、意大利文、
  俄文和阿拉伯文的内置 UI 翻译。
- 为阿拉伯语、波斯语、希伯来语、乌尔都语等常见 RTL 语言环境添加方向推断。
- 为页眉、下拉菜单、移动端文档菜单、侧边栏、目录、页面操作和上一页/下一页导航添加
  RTL 感知行为。
- 添加侧边栏诊断，用于检测缺失条目、空的自动生成目录、重复分区和重复页面归属。
- 增强侧边栏生成，支持 `autogenerate.structure`、`autogenerate.depth`
  和 `autogenerate.exclude`。
- 将侧边栏生成拆分为解析、自动生成、索引和诊断模块。
- 简化文档首页路由：现在由 `docsBase` 控制文档渲染位置，设置
  `docsBase: '/docs'` 时，`/` 可以留给自定义首页。
- 移除单独的 `homepage` 配置选项。
- 移除 `source.contentRoot`；编辑链接和贡献者源码路径现在直接使用内容条目文件路径。
- 更新发布工作流，自动生成 GitHub 发布说明。
- 改进下拉菜单、本地搜索、移动端文档导航和页面操作的 ARIA 状态、焦点处理与键盘行为。
- 修复已部署页面中的页面操作 URL，避免生成的助手链接保留本地开发源。
- 改进移动端复制页面的剪贴板处理。
- 为页眉 logo 图片添加高加载优先级。
- 在 RTL 正文内容中保持代码、键盘输入和示例文本从左到右显示。
