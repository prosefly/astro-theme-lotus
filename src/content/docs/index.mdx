# Astro Theme Lotus

Lotus is an installable documentation theme for Astro v7 with a declarative
theme config, a docs-first content model, and shared MDX building blocks for
real project documentation.

It is designed for teams that want docs they can ship quickly without rewriting
the entire shell. Install `@prosefly/astro-theme-lotus`, configure the site in
`src/theme.config.ts`, write pages in `src/content/docs/`, and let the content
tree drive section navigation.

## First-release scope

- MDX content with authored guide, components, and references sections.
- Dark mode foundation with configurable light, dark, or system defaults.
- Configurable nav links, header actions, footer sections, accent palette,
  fonts, and radius.
- Content-driven subnav and section sidebars.

Deferred for later releases:

- Search.
- i18n.
- Versioned docs.
