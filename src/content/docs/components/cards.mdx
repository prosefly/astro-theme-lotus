import { Card, CardGrid } from '@prosefly/astro-components';

Use `CardGrid` and `Card` when a page needs a compact set of next-step links or
feature summaries without introducing a heavy dashboard layout.

## Import

```mdx
import { Card, CardGrid } from '@prosefly/astro-components';
```

## Example

<CardGrid>
  <Card title="Read the configuration guide" href="/docs/configuration/project/">
    Review theme settings, typography tokens, and navigation structure before
    customising the theme.
  </Card>
  <Card title="Keep examples short">
    Cards work best for one paragraph of context and a single destination or
    action.
  </Card>
</CardGrid>

## Props

### `CardGrid`

| Prop | Type | Default | Notes |
| --- | --- | --- | --- |
| None | — | — | Creates a one-column mobile grid that expands to two columns on medium screens. |

### `Card`

| Prop | Type | Default | Notes |
| --- | --- | --- | --- |
| `title` | `string` | required | Heading rendered at the top of the card. |
| `href` | `string` | optional | When provided, the card renders as an anchor with link styling. |

## Notes

Cards are intentionally lightweight. They work best for short summaries and a
single destination, not for dense dashboard-style content.
