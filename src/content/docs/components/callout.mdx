import { Badge, Callout } from '@prosefly/astro-components';

Use `Callout` for inline guidance that needs more structure than a blockquote.
Use `Badge` for short labels that reinforce status or intent inside prose and
component demos.

## Import

```mdx
import { Badge, Callout } from '@prosefly/astro-components';
```

## Example

<Callout type="note" title="Document the default behavior">
  The note variant is a safe default for implementation details, caveats, and
  editorial guidance.
</Callout>

<Callout type="tip" title="Pair badges with compact labels">
  Combine `Badge` with headings, links, or card metadata when you want a small
  amount of emphasis without adding another container.
</Callout>

<div class="not-prose flex flex-wrap items-center gap-2">
  <span>Status badges:</span>
  <Badge>Neutral</Badge>
  <Badge color="accent">Accent</Badge>
  <Badge color="success">Stable</Badge>
  <Badge color="warning">Review</Badge>
</div>

## Props

### `Callout`

| Prop | Type | Default | Notes |
| --- | --- | --- | --- |
| `title` | `string` | variant label | Optional heading shown above the body copy. |
| `type` | `'note' \| 'tip' \| 'warning' \| 'danger'` | `'note'` | Changes border, background, and accent treatment. |

### `Badge`

| Prop | Type | Default | Notes |
| --- | --- | --- | --- |
| `variant` | `'solid' \| 'soft' \| 'subtle' \| 'outline' \| 'ghost'` | `'soft'` | Adjusts the badge visual treatment. |
| `color` | `'neutral' \| 'accent' \| 'success' \| 'warning' \| 'danger'` | `'neutral'` | Adjusts the badge tone while keeping the same shape and spacing. |

## Notes

Callouts are static content containers. They do not collapse, dismiss, or add
client-side behavior in the first release.
