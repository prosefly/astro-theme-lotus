# Badge

`Badge` is a compact inline label for status, maturity, and contextual metadata.
It is intentionally small so it can live inside paragraphs, headings, and card
content without changing the surrounding layout.

## Import

```mdx
import { Badge } from '@prosefly/astro-components';
```

## Example

<p>
  Components can ship in stages:
  {' '}
  <Badge variant="success">Released</Badge>
  {' '}
  <Badge variant="accent">New</Badge>
  {' '}
  <Badge variant="warning">Needs review</Badge>
</p>

## Props

| Prop | Type | Default | Notes |
| --- | --- | --- | --- |
| `variant` | `'neutral' \| 'accent' \| 'success' \| 'warning'` | `'neutral'` | Selects the badge colour treatment. |

## Notes

`Badge` is inline-only in the first release. Use it for status and metadata, not
as a replacement for larger navigation or filter controls.
