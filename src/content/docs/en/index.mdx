## Build The Shell First

Lotus provides the route integration, responsive docs chrome, sidebar model,
search index, Markdown source routes, and theme tokens. Start with the overview,
then configure the project settings that match your content structure.

## Explore The Pieces

- Use Configuration to set site identity, routing, i18n, navbar items, sidebars,
  page actions, and footer links.
- Use Essentials to review Markdown, frontmatter, images, search, and code block
  behavior.
- Use Components to compose callouts, steps, tabs, accordions, file trees,
  cards, badges, and icons inside MDX.
- Use References to tune typography and design tokens.
