Last updated: July 16, 2026

Astro Theme Lotus is a documentation theme. This demo site is designed to show
how Lotus pages, navigation, MDX components, and appearance settings work.

## Information we collect

This demo does not require an account and does not ask you to submit personal
information. If you contact the project maintainers through GitHub or another
external service, that service may collect information according to its own
privacy policy.

## Usage data

The demo can be deployed by different users in different environments. A
deployment may collect standard server logs such as request path, user agent,
referrer, IP address, and timestamp. Lotus itself does not add analytics or
tracking scripts by default.

## Cookies and local storage

Lotus stores the selected color mode in local storage so the site can preserve a
reader's light, dark, or system preference. This value stays in the browser and
is not sent to the Lotus project.

## Third-party services

This demo may link to GitHub, social platforms, package registries, and other
external websites. It may also load images or icon data from third-party
services when a page author chooses to use remote assets. Those services are
responsible for their own data practices.

## Changes

This policy may change as the demo evolves. Material updates should be reflected
by changing the date at the top of this page.

## Contact

For questions about this demo policy, open an issue in the project repository.
