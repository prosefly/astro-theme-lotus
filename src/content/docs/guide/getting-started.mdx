# Getting Started

Astro Theme Lotus is an installable documentation theme for Astro v7, Tailwind
CSS v4, and MDX.

## Install the packages

Install Lotus in an Astro project:

```sh
npm install @prosefly/astro-theme-lotus @prosefly/astro-components
```

## Add the integration

Configure Astro with the Lotus integration:

```ts
// astro.config.ts
import { defineConfig } from 'astro/config';
import lotus from '@prosefly/astro-theme-lotus';
import themeConfig from './src/theme.config';

export default defineConfig({
  integrations: [lotus(themeConfig)],
});
```

Register the docs content collection:

```ts
// src/content.config.ts
export { collections } from '@prosefly/astro-theme-lotus/content';
```

## Add content

Create MDX pages in `src/content/docs/`:

```txt
src/content/docs/
  index.mdx
  guide/
    getting-started.mdx
```

Lotus injects the `/docs/[...slug]` route, so `index.mdx` renders at `/docs/`.

## Start local development

Lotus uses the standard Astro development workflow:

```sh
npm run dev
```

This starts the local server so you can edit docs content, update
`src/theme.config.ts`, and preview MDX components in real pages.

## Core scripts

Keep the script surface intentionally small:

```sh
npm run dev
npm run check
npm run build
```

- `npm run dev` starts the Astro development server.
- `npm run check` runs `astro check` to validate content collections, MDX
  imports, and TypeScript.
- `npm run build` renders the production static site.

## Where to begin

1. Update `src/theme.config.ts` with your title, description, navigation, and
   footer links.
2. Replace or expand the example docs in `src/content/docs/`.
3. Reuse the shared MDX components from `@prosefly/astro-components` for callouts,
   cards, steps, badges, and tabs.
