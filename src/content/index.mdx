## Documentation Shell

Lotus provides the page chrome, docs route, search index, Markdown source
routes, and responsive navigation model for Astro documentation sites. Install
the integration, configure `src/theme.config.ts`, and let the content tree drive
the subnav, sidebar, table of contents, and previous/next links.

## What It Includes

- An Astro integration that mounts docs pages from `docsBase`.
- MDX support with Expressive Code, package manager tabs, and image galleries.
- Configurable navbar, sidebars, footer links, page actions, favicon, and logo.
- Theme tokens for accent color, gray scale, dark mode, and radius.
- Overrideable theme components for search, header, page actions, and page nav.
- Shared `@prosefly/astro-components` for common documentation UI.

## Designed For Real Docs

The default content structure separates guides, essentials, customization,
components, and references. That keeps the first install useful while still
leaving the information architecture fully configurable.
